üìÅ File Storage Structure

Upload Directory:

/rental_system/uploads/properties/

File Naming Convention:

Files are renamed with unique identifiers to prevent conflicts:

property_[unique_id].[extension]

Example: property_60f5a1b23.456789.jpg

Database Storage:

Image filenames are stored as JSON array in the images column:

["property_60f5a1b23.456789.jpg", "property_60f5a1c45.678901.png"]
